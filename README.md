# Open-Source Presentations

This repository contains technical presentations delivered at open-source events in France. These resources are made available to the community to share knowledge and promote the adoption of open-source solutions in industry.

### 1. IIoT and Open Source Experience (OSXP)

**Event:** [Open Source Experience](https://www.opensource-experience.com/en/)  
**Topic:** Open-Source in Industrial Automation (Make or Buy)

This presentation explores the challenges and opportunities of adopting open-source solutions in industrial automation and IIoT (Industrial Internet of Things). It covers:

- Advantages and challenges of choosing between internal development (Make) and external solutions (Buy)
- Available open-source ecosystems for Industry 4.0
- Experience feedback and best practices
- Technical and strategic considerations for integration

**Directory:** `/iiot-osxp/`

---

### 2. Introduction to ROS2

**Event:** [Campus du Libre](https://www.campus-du-libre.org/) - Lyon  
**Topic:** Introduction and presentation of the ROS2 framework

This presentation provides a comprehensive introduction to the ROS2 (Robot Operating System 2) framework, aimed at developers and robotics enthusiasts. Topics covered include:

- ROS2 fundamentals and architecture
- Key differences with ROS1
- Basic concepts: nodes, topics, services, actions
- Practical examples and use cases
- Ecosystem and development tools

**Directory:** `/ros2/`
